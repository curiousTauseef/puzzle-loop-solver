# Puzzle Loop Solver

Puzzle loop is a NPC problem.

## Usage

```
puzzle-loop-solver <puzzle file> <puzzle solution file>
```

## Puzzle Format

```
<cols> <rows>
<puzzle>

<comments>
```

## Cost

CPU: Ryzen R5 3500X

| Size  | Time |
| ----- | ---- |
| 40x50 | 30s  |
| 30x30 | 3s   |
